# T07 — Flutter: экраны групп

## Цель
Главный экран (список групп), создание группы, вступление по ссылке, страница группы.

## Экраны

### HomeScreen (`/home`)
- TabBar: «Мои группы» / «Профиль»
- Список карточек GroupCard: название, N участников, прогресс голосования, статус сезона
- FAB «Создать группу»
- Кнопка «Вступить по ссылке» (иконка в AppBar)
- Pull-to-refresh
- Пустое состояние с CTA

### CreateGroupScreen (`/groups/create`)
- Поле названия группы
- Мультивыбор категорий вопросов (чипы с эмодзи)
- Поле Telegram username (опционально, с `@` префиксом)
- Кнопка «Создать»
- После создания → показать BottomSheet с inviteUrl + кнопка «Поделиться»

### JoinGroupScreen (`/groups/join`)
- Поле ввода инвайт-кода или вставка ссылки целиком
- Превью группы (название, участники) после ввода (debounce 500ms)
- Кнопка «Вступить»

### GroupScreen (`/groups/:id`)
- AppBar: название группы, иконка share (инвайт), кнопка Telegram (если настроен)
- Статус активного сезона: прогресс-бар «X из N проголосовали»
- Кнопка «Проголосовать» (если не голосовал) / «Ждём пятницы» (если уже)
- Секция «Участники»: аватары + никнеймы, стрик-бейджи
- Кнопка «Открыть в Telegram» (если telegramUsername задан)

### Deeplink обработка
При переходе по `https://repa.app/join/{code}`:
1. Если не авторизован → сохранить код → auth → JoinGroupScreen
2. Если авторизован → сразу JoinGroupScreen с предзаполненным кодом
