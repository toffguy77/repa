# T13 — Flutter: Reveal Screen и анимации

## Цель
Главный экран продукта — анимированное открытие карточки репутации.

## Экраны

### RevealScreen (`/groups/:groupId/reveal`)

**Состояния:**
1. `waiting` — сезон ещё VOTING, показывает таймер до пятницы + прогресс группы
2. `ready` — сезон REVEALED, показывает кнопку «Открыть репу»
3. `opening` — анимация открытия
4. `revealed` — полная карточка

**Анимация открытия (состояние `opening`):**
- Тёмный фон
- Пульсирующий эмодзи 🍆 в центре (2 сек)
- Появление карточки снизу (slide up + fade in)
- Конфетти сверху (`flutter_animate` эффект)
- Общая длительность: 3 секунды

**Revealed состояние — карточка:**
```
┌─────────────────────┐
│  🍆 РЕПА            │
│                     │
│    [аватар эмодзи]  │
│    @username        │
│    "Хранитель тайн" │ ← репутационный титул
│                     │
│  🔥 Убежит при пожаре  │
│  ████████░░  67%    │
│                     │
│  🤫 Знает секреты   │
│  ██████░░░░  52%    │
│                     │
│  😂 Всех рассмешит  │
│  ████░░░░░░  38%    │
│                     │
│  [■■■■░░] скрыто    │ ← blur если есть ещё атрибуты
│                     │
│  ✨ Новые ачивки:   │
│  👑 Легенда группы  │
│                     │
│  [Поделиться] [Детектор] │
└─────────────────────┘
```

**Кнопки:**
- «Поделиться» — нативный share sheet с PNG карточкой
- «Детектор» — открывает DetectorBottomSheet
- «Открыть скрытые» (если есть) — спрашивает 5 кристаллов

### DetectorBottomSheet
- Заголовок: «Кто голосовал за тебя»
- Если не куплен: список участников заблюрен, кнопка «Узнать за 10 💎»
- Если куплен: список участников с аватарами
- ⚠️ Подсказка: «Детектор показывает кто участвовал в голосовании, но не как именно ответил»

### MembersRevealScreen (`/groups/:groupId/reveal/members`)
- Горизонтальный список карточек других участников (только топ-3 атрибута)
- Tap на карточку → модалка с полной информацией участника

### AchievementsUnlockScreen
Показывается автоматически если есть новые ачивки:
- Тёмный оверлей
- Анимация появления ачивки (bounce effect)
- Название и описание
- Кнопка «Далее» (если несколько) / «Закрыть»

## Анимации (flutter_animate)
```dart
// Появление карточки
Card().animate()
  .slideY(begin: 1, end: 0, duration: 600.ms, curve: Curves.easeOutCubic)
  .fadeIn(duration: 400.ms)

// Прогресс-бар атрибута
LinearProgressIndicator().animate()
  .custom(builder: (context, value, child) => ...) // анимация от 0 до percentage
  .delay(index * 200.ms)

// Ачивка unlock
AchievementCard().animate()
  .scale(begin: Offset(0.5, 0.5), duration: 400.ms, curve: Curves.elasticOut)
```

## Критерии готовности
- [ ] Анимация открытия плавная (60fps)
- [ ] Все три состояния экрана работают
- [ ] Шеринг PNG открывает нативный share sheet
- [ ] Детектор корректно показывает/скрывает данные
- [ ] Карточки других участников доступны только из приложения
