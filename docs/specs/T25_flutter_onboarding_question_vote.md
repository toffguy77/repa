# T25 — Flutter: онбординг и экран голосования за вопросы

## Цель
Онбординг нового пользователя (3 экрана) и экран голосования за вопросы следующего сезона.

## 1. Онбординг

### Когда показывать
- Только при `isNew: true` после регистрации (сразу после ProfileSetupScreen)
- Флаг `onboarding_done` в flutter_secure_storage — больше не показывать

### OnboardingScreen (`/onboarding`)
Три слайда с PageView. Кнопка «Пропустить» всегда видна. Последний слайд — «Начать».

**Слайд 1 — Группа**
```
🍆
Создай группу
для своих

Добавь класс, компанию
или просто друзей
по ссылке-инвайту
```

**Слайд 2 — Голосование**
```
🗳
Голосуй анонимно

Каждую неделю — смешные
вопросы про участников.
Никто не узнает твой ответ.
```

**Слайд 3 — Reveal**
```
🎭
Узнай в пятницу

В 20:00 — твоя карточка
репутации. Поделись
с чатом или оставь себе.
```

### Дизайн слайдов
- Тёмный фон (`Color(0xFF1A0533)`)
- Крупный эмодзи (96px)
- Белый заголовок (24px bold)
- Серый подзаголовок (16px)
- Индикатор страниц снизу (3 точки)
- Анимация смены слайда: fade + slide

### После онбординга
→ HomeScreen с empty state (предложение создать группу или вступить)

## 2. Экран голосования за вопросы следующей недели

### QuestionVoteScreen (`/groups/:groupId/question-vote`)
Открывается по deeplink `screen: 'question-vote'` из пуша в воскресенье,
или из GroupScreen (кнопка видна с воскресенья по понедельник).

**Структура:**
- Заголовок: «Какой вопрос добавить на следующей неделе?»
- Подзаголовок: «Победивший вопрос войдёт в ротацию»
- Три карточки вопросов (получить из GET /groups/:id/next-season/question-candidates)
- Tap на карточку → выделение + POST /groups/:id/next-season/vote-question
- После голосования: «Готово! Узнаешь результат в понедельник» + счётчик голосов

**QuestionCandidateCard:**
```dart
// Карточка вопроса-кандидата:
// - Эмодзи категории
// - Текст вопроса
// - Кол-во голосов (обновляется после своего голосования)
// Selected state: purple border + checkmark
// Tap недоступен если уже голосовал (один голос на неделю)
```

**Состояния экрана:**
- `loading` — skeleton
- `voting` — три карточки, можно выбрать
- `voted` — карточки disabled, показан свой выбор + счётчики всех
- `unavailable` — голосование не открыто (показать когда откроется)

### Когда доступен экран
Голосование за вопросы открыто с воскресенья 12:00 по понедельник 17:00 МСК.
Вне этого окна — показать сообщение «Голосование откроется в воскресенье в полдень».

### Кнопка в GroupScreen
```dart
// Показывать только в окне ВС 12:00 — ПН 17:00
// Если пользователь ещё не голосовал: пульсирующая кнопка «Выбери вопрос недели»
// Если уже голосовал: «Ты выбрал вопрос ✓» (серая, неактивная)
```

## Критерии готовности
- [ ] Онбординг показывается только новым пользователям
- [ ] «Пропустить» работает на любом слайде
- [ ] Флаг в secure storage предотвращает повторный показ
- [ ] QuestionVoteScreen открывается по deeplink
- [ ] Голосование недоступно вне временного окна
- [ ] После голосования — счётчики других обновляются
- [ ] Кнопка в GroupScreen видна только в нужное время
